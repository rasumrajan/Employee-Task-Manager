#  Employee Task Manager & Productivity Tracker

A professional **Django-based Web Application** designed for managers to assign tasks, set deadlines, and track real-time employee productivity.

##  Project Overview
The goal of this project is to eliminate communication gaps between a Boss and their Employees. It provides a transparent system to monitor how much time is taken for each task and ensures deadlines are met.

##  Core Features
- **Task Assignment:** Admin/Boss can create tasks and assign them to specific employees.
- **Deadline Management:** Every task has a fixed deadline to ensure timely completion.
- **Time Tracking:** Real-time logging of the duration spent by an employee on a particular task.
- **Status Updates:** Employees can update their progress (Pending, In Progress, Completed) from their dedicated dashboard.
- **Role-Based Access:** Secure login for both Admin and Staff with different permission levels.

##  Technical Implementation
- **Backend:** Python 3.12 / Django 5.x
- **Database:** SQLite (Development) / PostgreSQL (Production)
- **Frontend:** HTML5, CSS (Bootstrap 5), and Vanilla JavaScript for time-tracking logic.
- **Authentication:** Django Built-in User Auth System.
